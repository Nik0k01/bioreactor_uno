# Komendy

#### Ramka szybka 

```sh
DEBUG_FAST,{on/off}
```
**on/off** - **0**(wyłączone) lub **1**(włączone)

Przykład:
```sh
DEBUG_FAST,1 // Włączenie ramki szybkiej
```
